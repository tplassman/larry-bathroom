# Bathroom Remodel Task List

[Materials List](https://docs.google.com/spreadsheets/d/1Uye_KnBVwiA66lSWBx43uSiD5_YzCsYk-MWyENiruzA/edit?gid=0#gid=0)

### Before start
- [ ] Gather tools, rent concrete saw & rebar scanner, pick up drain fittings and 2" ABS pipe
- [ ] Demo closet — remove rod, shelving, drywall on shared wall; expose studs
- [ ] Scan slab for rebar with rental scanner; mark drain location (21.5" from back wall, centered on 48")
- [ ] Mark wall opening for closet breakthrough; confirm no load-bearing issues
- [ ] Open wall between bathroom and closet — cut drywall, remove studs as needed for 48" opening along left wall
- [ ] Access crawlspace — locate main drain, confirm depth and material, plan tie-in method

### Day 1
- [ ] Layout and mark concrete cut lines; snap chalk lines for drain trench path to crawlspace
- [ ] Concrete saw cuts along marked lines (shallow first pass at 2", then full depth) — go slow, watch for rebar; stop and assess if resistance changes
- [ ] Break out concrete with sledgehammer and chisels; remove debris
- [ ] Excavate trench to crawlspace; hand-dig at wall penetration point
- [ ] Clean trench, verify slope with level and tape measure
- [ ] Dry-fit drain pipe run from shower location to crawlspace tie-in; mark cuts

### Day 2
- [ ] Glue up drain run — P-trap, drain riser, horizontal run through trench to crawlspace; leave drain riser stub 1–2" above slab, cap it
- [ ] Crawlspace work — tie into main drain line, glue connections, verify slope one final time
- [ ] Install AAV in interstitial ceiling space or trace/confirm vanity vent situation
- [ ] Water test — plug drain, fill with water, check crawlspace for leaks; let sit 30 min
- [ ] Pack trench with gravel bed under pipe; backfill and tamp around pipe
- [ ] Mix and pour concrete patch around drain and trench; smooth flush with existing slab; leave drain stub protruding — do not cover

### Day 3
- [ ] Frame shower alcove — install new studs to exact Ovation Curve spec (7-3/4" / 10-5/16" spacing)
- [ ] Frame closet side walls and back wall of shower alcove; check square and plumb repeatedly
- [ ] Install blocking for shower valve and shower arm at correct heights
- [ ] Rough-in shower valve supply lines (hot and cold); cap and pressure test
- [ ] Cut OSB panels to size (back wall 26-1/4" x 68-7/8", side walls 17" x 68-7/8")
- [ ] Install OSB panels — 2" above future base deck, screw flush to studs; mark plumbing cutouts
- [ ] Cut plumbing holes in OSB; install 1x4 door support if adding shower door

### Day 4
- [ ] Verify concrete patch is fully cured and floor is level; shim if needed
- [ ] Install drain body into base before setting; apply silicone sealant at drain
- [ ] Set base into alcove with two people; level in both directions; attach 8 clips to studs
- [ ] Dry-fit both side wall panels; check plumb, mark plumbing hole locations; drill holes
- [ ] Dry-fit rear wall panel between side panels; check for gaps; adjust shims as needed
- [ ] Mark all panel positions; remove panels and prep for adhesive day

### Day 5
- [ ] Apply adhesive to side wall OSB (2 tubes each); reinstall side panels to marked positions; screw flanges
- [ ] Install extruded wall connectors on inner edges of both side panels; hand-tighten only
- [ ] Apply adhesive to rear OSB (4 tubes); snap rear panel into connectors; screw top flange into studs
- [ ] Apply bracing across panels to hold contact with adhesive; tape base-to-panel joint

### Day 6
- [ ] Remove bracing; apply 100% silicone sealant to all panel joints and base-to-wall seam
- [ ] Install shower valve trim, shower arm, and showerhead
- [ ] Install drain cover; run water to test drain flow and check for leaks at all connections
- [ ] Install shower door if using (AM00845400) per manufacturer instructions
- [ ] Patch and paint bathroom walls around new opening; install trim at wall transition
- [ ] Final cleanup; let sealant cure 24 hrs before first use
